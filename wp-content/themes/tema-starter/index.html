Introduction

Starter is so clean and supper flexible One Page Responsive Theme. Starter Bulit with fully functional Theme such as slider, team, clients Post Type with modern design trend. It has Six awesome presets with great color combinations.

If you are looking for a fully responsive ready theme to fit your next One Page site then Starter is the best choice for you.

Getting Started

Wordpress Installation

This Theme has been developed only for Wordprss 3.5 or Higher. So, you need to install wordpress fast then use this theme. Please see the wordpress installation process . Here is the documentation Wordpress Codex - Installing Wordpress

Wordpress Theme Installation

Theme installation is very easy process. First you need to upload theme files and active it. The Theme file can be uploaded two ways

I. FTP Upload: Using your favourite FTP program . Frist you have to unzip theme files then upload into the folder path /wp-content/themes/ on your server and active it from admin panel.

II. Wordpress Upload: From Wordpress Admin Panel go the following path Appearance->themes->Install-themes->upload and select the zip theme . Click install Now‚Äù and theme will be uploaded and installed.

Starter Theme Installation

Install Starter Theme by below steps

starter Installation

Fig: Install starter Step by Step


Step 1: In Dashboard click on Dasboard => Appearance Menu

Step 2: Then click on Dasboard => Appearance => Theme Menu

Step 3: Then click on Install Themes tab.

Step 4: Then click on Upload menu.

Step 5: Then click on Choose file and select starter.zip file from your computer directory.

Step 6: Now click on Install Now button and wait for the installation complete. After successful installation of theme click on active theme click and enjoy theme theme feature.

starter Essential Plugins and Installation

For using all Features you have to install and active all required plugins. All Essential plugins are included in the starter theme.

  • starter Slider
  • starter Team
  • starter Client

Install above essential plugins by easy 2 step.

Fig: Essential plugins installation

Step 1: In Dashboard, click on Dasboard => Appearance Menu => Install Plugins Menu

Step 2: Then click on Install link under plugin title and enjoy the plugins.

Setup Permalink(Required)

Make sure to configure permalink. From admin panel go to the following path Settings->permalik then select (Post name).

Demo Import

Import Sample Data: If you want to use sample data for your website then you need to install sample date. See the way how to install Import Sample Data tools->import->wordpress Bottom of the page have wordpress . Click wordpress then you can see the Install Now

Widgets Setting Import

Import Widget Setting Data: If you want to use Demo site widget setting for your website. Tools->Widgets Setting Import Select file from Wigdets-settings Folder From Package Folder. Then Click on Show Widget Settings Button.

Set Home Page(Required)

Set Static Home Page(Required): WordPress shows blog posts by default as the front page. If you want to look like demo starter then you need to set Home Page. See the way how to set Static Hope Page Settings->reading . From here select static page then for Front page select home

How to Create Page

Create Default Page:

Create page as default page. Change page seeting from Page Settings:

  • Page Section Type as "Default"
  • You can use section Background color from color Picker.
  • You can disiable page from menu from "Disable From Menu" section
  • You Can Disable Title

Create Parallax Page:

Change page seeting from Page Settings:

  • Page Section Type as "Parallax"
  • You can use section Background color from color Picker.
  • You can disiable page from menu from "Disable From Menu" section
  • You Can Disable Title
  • Add parallax Background Url

Create Full-Width Page:

Change page seeting from Page Settings:

  • Page Section Type as "Full-Width"
  • You can use section Background color from color Picker.
  • You can disiable page from menu from "Disable From Menu" section
  • You Can Disable Title
